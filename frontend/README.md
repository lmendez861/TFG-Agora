# Frontend

Interfaz web para la aplicación de gestión de empresas colaboradoras.

## Objetivo
Diseñar una UI limpia que permita al centro educativo gestionar:
- Empresas colaboradoras
- Tutores profesionales y académicos
- Estudiantes asignados
- Convenios y periodos de prácticas
- Seguimiento y evaluación

## Próximos pasos
1. Definir wireframes y componentes principales (dashboard, listados, fichas).
2. Seleccionar la tecnología frontend (mantener HTML/CSS/JS o evolucionar a un framework).
3. Conectar con la API REST del backend.
